# mdx格式示例

# Markdown 功能

Docusaurus 支持 **[Markdown](https://daringfireball.net/projects/markdown/syntax)** 和一些 **附加特性**。

## 前置元数据

Markdown 文档顶部有称为 [前置元数据](https://jekyllrb.com/docs/front-matter/) 的元数据：
```text title="my-doc.md"
// highlight-start
---
id: my-doc-id
title: 我的文档标题
description: 我的文档描述
slug: /my-custom-url
---
// highlight-end
```

## 提示和警告

Docusaurus 有一种特殊的语法来创建提示和警告：

```md
:::tip My tip

使用这个超棒的功能选项

:::

:::danger 小心

这个动作很危险

:::
```

:::tip My tip

使用这个超棒的功能选项!

:::

:::danger 小心

这个动作很危险!

:::

## MDX 和 React 组件

[MDX](https://mdxjs.com/) 可以使你的文档更具 **交互性** 并且允许在 **Markdown** 中使用任何 **React 组件**：
```jsx
export const Highlight = ({children, color}) => (
  <span
    style={{
      backgroundColor: color,
      borderRadius: '20px',
      color: '#fff',
      padding: '10px',
      cursor: 'pointer',
    }}
    onClick={() => {
      alert(`您点击了颜色 ${color}，标签为 ${children}`);
    }}>
    {children}
  </span>
);

这是 <Highlight color="#25c2a0">Docusaurus 绿</Highlight> !

这是 <Highlight color="#1877F2">Facebook 蓝</Highlight> !
```
