# Markdown 教程
基础 Markdown 语法教学

## 1. 标题
```
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题
```

# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题

## 2. 样式
```
*斜体文字* 或 _斜体文字_
**粗体文字** 或 __粗体文字__
***粗斜体文字*** 或 ___粗斜体文字___
~~删除线~~
<u>下划线</u>
```

*斜体文字* 或 _斜体文字_

**粗体文字** 或 __粗体文字__

***粗斜体文字*** 或 ___粗斜体文字___

~~删除线~~

<u>下划线</u>
